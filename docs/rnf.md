# 🛠️ Requisitos Não Funcionais (RNF)

Requisitos de qualidade, segurança e desempenho necessários para a operação da clínica.

- [x] [cite_start]**RNF01 - Segurança de Dados (LGPD)**: Todos os prontuários e dados de saúde devem ser criptografados, seguindo as normas de sigilo médico-paciente[cite: 205].
- [x] **RNF02 - Disponibilidade**: O sistema de agendamento deve estar disponível 24/7 para que o paciente marque consultas a qualquer momento.
- [x] **RNF03 - Performance**: O carregamento do calendário de vagas não deve ultrapassar 2 segundos para evitar desistências.
- [x] [cite_start]**RNF04 - Multiplataforma**: O app deve ser responsivo, funcionando perfeitamente em diferentes tamanhos de tela de smartphones[cite: 217].
- [x] [cite_start]**RNF05 - Backup**: Sincronização automática em nuvem (Firebase) para garantir que o dentista não perca a agenda em caso de falha no dispositivo[cite: 208].
