# 🛠️ Requisitos Não Funcionais (RNF)

Requisitos de qualidade, segurança e desempenho necessários para a operação da clínica.

- [ ] **RNF01 - Disponibilidade**: O sistema de agendamento deve estar disponível 24/7 para que o paciente marque consultas a qualquer momento.
- [ ] **RNF02 - Performance**: O carregamento do calendário de vagas não deve ultrapassar 2 segundos para evitar desistências.
- [ ] **RNF03 - Multiplataforma**: O app deve ser responsivo, funcionando perfeitamente em diferentes tamanhos de tela de smartphones.
